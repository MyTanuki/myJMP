# บ้านพักดี — ระบบบริหารหอพัก 🏠

เว็บแอปบริหารจัดการห้องพัก (อ้างอิงฟังก์ชันจาก horganice.in.th แต่ปรับธีมเป็นโทน ฟ้า/เขียว/แดง, สะอาด, responsive)

## ฟีเจอร์
- **ภาพรวม (Dashboard)** — จำนวนห้อง อัตราเข้าพัก รายได้/ค้างชำระเดือนปัจจุบัน ทางลัด
- **ห้องพัก** — เพิ่ม/แก้ไข/ลบห้อง ดูสถานะว่าง–มีผู้เช่า ค่าเช่า
- **ผู้เช่า / สัญญา** — ข้อมูลผู้เช่า เงินมัดจำ วันเข้าพัก ช่วงสัญญา แจ้งย้ายออก
- **บิล / มิเตอร์** — จดเลขมิเตอร์น้ำ–ไฟต่อเดือน คำนวณค่าใช้จ่ายอัตโนมัติ ออกบิล ทำเครื่องหมายชำระ เลื่อนดูได้รายเดือน
- **ตั้งค่า** — ชื่อหอ อัตราค่าน้ำ–ค่าไฟ (ใช้เป็นค่าเริ่มต้นตอนออกบิล)

## เทคโนโลยี
Next.js 16 (App Router) · React 19 · Tailwind CSS 4 · Prisma 7 + SQLite · JWT auth (jose) ผ่าน httpOnly cookie

## เริ่มใช้งาน

```bash
npm install
npx prisma migrate dev   # สร้างฐานข้อมูล
npm run db:seed          # ใส่ข้อมูลตัวอย่าง
npm run dev
```

เปิด http://localhost:3000

**บัญชีตัวอย่าง:** `demo@dorm.test` / `123456`
(หรือกด “สมัครใหม่” เพื่อสร้างหออื่น)

## โครงสร้าง
```
src/
  app/
    login/            หน้าเข้าสู่ระบบ / สมัคร (+ server actions)
    (app)/            หน้าหลังบ้าน (ต้องล็อกอิน)
      page.tsx        Dashboard
      rooms/          จัดการห้อง
      tenants/        จัดการผู้เช่า
      invoices/       บิล/มิเตอร์
      settings/       ตั้งค่า
  components/         Sidebar, Modal, UI primitives
  lib/                db (Prisma), auth (JWT), format (เงิน/วันที่/คำนวณบิล)
prisma/
  schema.prisma       โครงสร้างข้อมูล
  seed.ts             ข้อมูลตัวอย่าง
```
